Various exercises implementing functions of the OpenMP framework.
* omp1.c: Parallel computation for large number of loops with multiple threads.
* omp2.c: Parallel image blurring with loops or tasks.
* omp3.c: Simple summing with taskloop.
* cuinfo.c: Prints specs for CUDA device.
* gaussian-blur.c: Parallel image blurring through GPU offloading using CUDA.
